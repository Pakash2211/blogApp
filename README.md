<h1>blogApp</h1>

<h2>A Solo project built by me within 1 day.</h2>
<br>

## Features :

 <p>--> Create-blog</p>
 <p>--> Edit blog</p>
 <p>--> Delete blog</p>
 <p>--> View created blog</p>
 <p>--> Search blog </p>



## Tech Stack

- HTML 5
- CSS 3
- Javascript
- JSON Server


Start the server
```bash
  json-server --watch db2.json
```
